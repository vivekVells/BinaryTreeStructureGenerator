# BinaryTreeStructureGenerator
Objective:
- To generate binary tree structuer for any given integer array

> Say arr[] = { 7, 4, 9, 2, 5, 8, 12} with first arr[0] as root, then the output would be 
```
                   7
                /     \
               4       9
              / \     / \
             2   5   8   12
```
